# 🚀 Варианты деплоя с автоматическим импортом

Для работы автоматического импорта треков из YouTube/VK/Spotify нужен **ffmpeg** и **yt-dlp**.

## ✅ Вариант 1: Railway.app (РЕКОМЕНДУЮ)

**Преимущества:**
- ✅ Бесплатный план ($5 кредитов/месяц)
- ✅ ffmpeg работает из коробки
- ✅ Простая настройка (5 минут)
- ✅ Автодеплой из GitHub

**Инструкция:** см. `RAILWAY_DEPLOY.md`

**Стоимость:** Бесплатно (до $5/месяц)

## 🏠 Вариант 3: Свой VPS (для продакшена)

**Преимущества:**
- ✅ Полный контроль
- ✅ Нет ограничений
- ✅ Можно хранить много треков

**Недостатки:**
- ❌ Нужно настраивать самому
- ❌ Нужно следить за безопасностью

**Подходящие провайдеры:**
- DigitalOcean ($4-6/месяц)
- Hetzner ($3-5/месяц)
- Contabo ($4-7/месяц)

**Быстрая установка на Ubuntu:**
```bash
# Установка Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Установка ffmpeg и yt-dlp
sudo apt-get install -y ffmpeg python3-pip
pip3 install yt-dlp

# Клонирование и запуск
git clone <your-repo>
cd backend
npm install
node server.js
```
